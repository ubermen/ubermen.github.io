25 class BigQueryGetDataOperator(BaseOperator):
26  """
27  Fetches the data from a BigQuery table (alternatively fetch data for selected columns)
28  and returns data in a python list. The number of elements in the returned list will
29  be equal to the number of rows fetched. Each element in the list will again be a list
30  where element would represent the columns values for that row.
31 
32  **Example Result**: ``[['Tony', '10'], ['Mike', '20'], ['Steve', '15']]``
33 
34  .. note::
35  If you pass fields to ``selected_fields`` which are in different order than the
36  order of columns already in
37  BQ table, the data will still be in the order of BQ table.
38  For example if the BQ table has 3 columns as
39  ``[A,B,C]`` and you pass 'B,A' in the ``selected_fields``
40  the data would still be of the form ``'A,B'``.
41 
42  **Example**: ::
43 
44  get_data = BigQueryGetDataOperator(
45  task_id='get_data_from_bq',
46  dataset_id='test_dataset',
47  table_id='Transaction_partitions',
48  max_results='100',
49  selected_fields='DATE',
50  bigquery_conn_id='airflow-service-account'
51  )
52 
53  :param dataset_id: The dataset ID of the requested table. (templated)
54  :type destination_dataset_table: string
55  :param table_id: The table ID of the requested table. (templated)
56  :type table_id: string
57  :param max_results: The maximum number of records (rows) to be fetched
58  from the table. (templated)
59  :type max_results: string
60  :param selected_fields: List of fields to return (comma-separated). If
61  unspecified, all fields are returned.
62  :type selected_fields: string
63  :param bigquery_conn_id: reference to a specific BigQuery hook.
64  :type bigquery_conn_id: string
65  :param delegate_to: The account to impersonate, if any.
66  For this to work, the service account making the request must have domain-wide
67  delegation enabled.
68  :type delegate_to: string
69  """
