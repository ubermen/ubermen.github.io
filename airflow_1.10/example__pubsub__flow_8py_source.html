21 This example DAG demonstrates how the PubSub*Operators and PubSubPullSensor

23 
24 NOTE: project_id must be updated to a GCP project ID accessible with the
25  Google Default Credentials on the machine running the workflow

65 echo_template = '''
66 {% for m in task_instance.xcom_pull(task_ids='pull-messages') %}
67  echo "AckID: {{ m.get('ackId') }}, Base64-Encoded: {{ m.get('message') }}"
