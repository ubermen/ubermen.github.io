26  """
27  Execute tasks (commands) on QDS (https://qubole.com).
28 
29  :param qubole_conn_id: Connection id which consists of qds auth_token
30  :type qubole_conn_id: str
31 
32  kwargs:
33  :command_type: type of command to be executed, e.g. hivecmd, shellcmd, hadoopcmd
34  :tags: array of tags to be assigned with the command
35  :cluster_label: cluster label on which the command will be executed
36  :name: name to be given to command
37  :notify: whether to send email on command completion or not (default is False)
38 
39  **Arguments specific to command types**
40 
41  hivecmd:
42  :query: inline query statement
43  :script_location: s3 location containing query statement
44  :sample_size: size of sample in bytes on which to run query
45  :macros: macro values which were used in query
46  prestocmd:
47  :query: inline query statement
48  :script_location: s3 location containing query statement
49  :macros: macro values which were used in query
50  hadoopcmd:
51  :sub_commnad: must be one these ["jar", "s3distcp", "streaming"] followed by
52  1 or more args
53  shellcmd:
54  :script: inline command with args
55  :script_location: s3 location containing query statement
56  :files: list of files in s3 bucket as file1,file2 format. These files will be
57  copied into the working directory where the qubole command is being
58  executed.
59  :archives: list of archives in s3 bucket as archive1,archive2 format. These
60  will be unarchived intothe working directory where the qubole command is
61  being executed
62  :parameters: any extra args which need to be passed to script (only when
63  script_location is supplied)
64  pigcmd:
65  :script: inline query statement (latin_statements)
66  :script_location: s3 location containing pig query
67  :parameters: any extra args which need to be passed to script (only when
68  script_location is supplied
69  sparkcmd:
70  :program: the complete Spark Program in Scala, SQL, Command, R, or Python
71  :cmdline: spark-submit command line, all required information must be specify
72  in cmdline itself.
73  :sql: inline sql query
74  :script_location: s3 location containing query statement
75  :language: language of the program, Scala, SQL, Command, R, or Python
76  :app_id: ID of an Spark job server app
77  :arguments: spark-submit command line arguments
78  :user_program_arguments: arguments that the user program takes in
79  :macros: macro values which were used in query

81  :db_tap_id: data store ID of the target database, in Qubole.
82  :query: inline query statement
83  :macros: macro values which were used in query

85  :mode: 1 (simple), 2 (advance)
86  :hive_table: Name of the hive table

88  :dbtap_id: data store ID of the target database, in Qubole.
89  :db_table: name of the db table
90  :db_update_mode: allowinsert or updateonly
91  :db_update_keys: columns used to determine the uniqueness of rows
92  :export_dir: HDFS/S3 location from which data will be exported.
93  :fields_terminated_by: hex of the char used as column separator in the dataset
94  dbimportcmd:
95  :mode: 1 (simple), 2 (advance)
96  :hive_table: Name of the hive table
97  :dbtap_id: data store ID of the target database, in Qubole.
98  :db_table: name of the db table
99  :where_clause: where clause, if any
100  :parallelism: number of parallel db connections to use for extracting data
101  :extract_query: SQL query to extract data from db. $CONDITIONS must be part
102  of the where clause.
103  :boundary_query: Query to be used get range of row IDs to be extracted
104  :split_column: Column used as row ID to split data into ranges (mode 2)
105
