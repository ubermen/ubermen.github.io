43  """
44  Create new connection to Segment
45  and allows you to pull data out of Segment or write to it.
46 
47  You can then use that file with other
48  Airflow operators to move the data around or interact with segment.
49 
50  :param segment_conn_id: the name of the connection that has the parameters

52  The connection should be type `json` and include a
53  write_key security token in the `Extras` field.
54  :type segment_conn_id: str
55  :param segment_debug_mode: Determines whether Segment should run in debug mode.
56  Defaults to False
57  :type segment_debug_mode: boolean
58  .. note::
59  You must include a JSON structure in the `Extras` field.
60  We need a user's security token to connect to Segment.
61  So we define it in the `Extras` field as:
62  `{"write_key":"YOUR_SECURITY_TOKEN"}`
63  """

73  if self.write_key is None:
