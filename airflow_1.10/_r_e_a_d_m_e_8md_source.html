1 ## Variable Editor
2 ----
3 This folder contains forms used to edit values in the "Variable" key-value
4 store. This data can be edited under the "Admin" admin tab, but sometimes
5 it is preferable to use a form that can perform checking and provide a nicer
6 interface.
7 
8 ### Adding a new form
9 
10 1. Create an html template in `templates/variables` folder
11 2. Provide an interface for the user to provide input data
12 3. Submit a post request that adds the data as json.
13 
14 An example ajax POST request is provided below:
15 ```js
16 $("#submit-btn").click(function() {
17  form_data = getData()
18  if (isValid(form_data)) {
19  $.ajax({
20  method: "POST",
21  url: "backfill",
22  data: JSON.stringify(form_data),
23  dataType: "json",
24  contentType: "application/json",
25  success: function(response_data) {
26  console.log("success.")
27  },
28  failure: function(response_data) {
29  console.log("post error.")
30  }
31  })
32  }
33  else {
34  console.log("input error.")
35  }
36 });
37 ```
