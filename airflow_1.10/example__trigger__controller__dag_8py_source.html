19 """This example illustrates the use of the TriggerDagRunOperator. There are 2
20 entities at work in this scenario:
21 1. The Controller DAG - the DAG that conditionally executes the trigger
22 2. The Target DAG - DAG being triggered (in example_trigger_target_dag.py)
23 
24 This example illustrates the following features :
25 1. A TriggerDagRunOperator that takes:
26  a. A python callable that decides whether or not to trigger the Target DAG
27  b. An optional params dict passed to the python callable to help in
28  evaluating whether or not to trigger the Target DAG
29  c. The id (name) of the Target DAG
30  d. The python callable can add contextual info to the DagRun created by
31  way of adding a Pickleable payload (e.g. dictionary of primitives). This
32  state is then made available to the TargetDag
33 2. A Target DAG : c.f. example_trigger_target_dag.py
34 """
