38  :param trigger_dag_id: the dag_id to trigger
39  :type trigger_dag_id: str
40  :param python_callable: a reference to a python function that will be
41  called while passing it the ``context`` object and a placeholder
42  object ``obj`` for your callable to fill and return if you want
43  a DagRun created. This ``obj`` object contains a ``run_id`` and
44  ``payload`` attribute that you can modify in your function.
