26 class GoogleCloudStorageCreateBucketOperator(BaseOperator):
27  """
28  Creates a new bucket. Google Cloud Storage uses a flat namespace,
29  so you can't create a bucket with a name that is already in use.
30 
31  .. seealso::
32  For more information, see Bucket Naming Guidelines:
33  https://cloud.google.com/storage/docs/bucketnaming.html#requirements
34 
35  :param bucket_name: The name of the bucket. (templated)
36  :type bucket_name: string
37  :param storage_class: This defines how objects in the bucket are stored
38  and determines the SLA and the cost of storage (templated). Values include
39 
40  - ``MULTI_REGIONAL``
41  - ``REGIONAL``
42  - ``STANDARD``
43  - ``NEARLINE``
44  - ``COLDLINE``.
45  If this value is not specified when the bucket is
46  created, it will default to STANDARD.
47  :type storage_class: string
48  :param location: The location of the bucket. (templated)
49  Object data for objects in the bucket resides in physical storage
50  within this region. Defaults to US.
51 
52  .. seealso::
53  https://developers.google.com/storage/docs/bucket-locations
54 
55  :type location: string
56  :param project_id: The ID of the GCP Project. (templated)
57  :type project_id: string
58  :param labels: User-provided labels, in key/value pairs.
59  :type labels: dict
60  :param google_cloud_storage_conn_id: The connection ID to use when
61  connecting to Google cloud storage.
62  :type google_cloud_storage_conn_id: string
63  :param delegate_to: The account to impersonate, if any.
64  For this to work, the service account making the request must
65  have domain-wide delegation enabled.
66  :type delegate_to: string
67 
68  **Example**:
69  The following Operator would create a new bucket ``test-bucket``
70  with ``MULTI_REGIONAL`` storage class in ``EU`` region ::
71 
72  CreateBucket = GoogleCloudStorageCreateBucketOperator(
73  task_id='CreateNewBucket',
74  bucket_name='test-bucket',
75  storage_class='MULTI_REGIONAL',
76  location='EU',
77  labels={'env': 'dev', 'team': 'airflow'},
78  google_cloud_storage_conn_id='airflow-service-account'
79  )
80  """
