25  """
26  Executes hql code in a specific Hive database.
27 
28  :param hql: the hql to be executed
29  :type hql: string
30  :param hive_cli_conn_id: reference to the Hive database
31  :type hive_cli_conn_id: string

33  ${var} gets translated into jinja-type templating {{ var }}. Note that

36  object documentation for more details.
37  :type hiveconf_jinja_translate: boolean
38  :param script_begin_tag: If defined, the operator will get rid of the
39  part of the script before the first occurrence of `script_begin_tag`
40  :type script_begin_tag: str
41  :param mapred_queue: queue used by the Hadoop CapacityScheduler
42  :type mapred_queue: string
43  :param mapred_queue_priority: priority within CapacityScheduler queue.
44  Possible settings include: VERY_HIGH, HIGH, NORMAL, LOW, VERY_LOW
45  :type mapred_queue_priority: string
46  :param mapred_job_name: This name will appear in the jobtracker.
47  This can make monitoring easier.
48  :type mapred_job_name: string
49  """
