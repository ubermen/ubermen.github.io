11 Airflow is a platform to programmatically author, schedule and monitor
12 workflows.
13 
14 When workflows are defined as code, they become more maintainable,
15 versionable, testable, and collaborative.
16 
17 Use Airflow to author workflows as directed acyclic graphs (DAGs) of tasks.
18 The Airflow scheduler executes your tasks on an array of workers while
19 following the specified dependencies. Rich command line utilities make
20 performing complex surgeries on DAGs a snap. The rich user interface
21 makes it easy to visualize pipelines running in production,
22 monitor progress, and troubleshoot issues when needed.

29 ## Beyond the Horizon
30 
31 Airflow **is not** a data streaming solution. Tasks do not move data from
32 one to the other (though tasks can exchange metadata!). Airflow is not
33 in the [Spark Streaming](http://spark.apache.org/streaming/)
34 or [Storm](https://storm.apache.org/) space, it is more comparable to
35 [Oozie](http://oozie.apache.org/) or
36 [Azkaban](https://azkaban.github.io/).
37 
38 Workflows are expected to be mostly static or slowly changing. You can think
39 of the structure of the tasks in your workflow as slightly more dynamic
40 than a database structure would be. Airflow workflows are expected to look
41 similar from a run to the next, this allows for clarity around
42 unit of work and continuity.

44 ## Principles
45 
46 - **Dynamic**: Airflow pipelines are configuration as code (Python), allowing for dynamic pipeline generation. This allows for writing code that instantiates pipelines dynamically.
47 - **Extensible**: Easily define your own operators, executors and extend the library so that it fits the level of abstraction that suits your environment.
48 - **Elegant**: Airflow pipelines are lean and explicit. Parameterizing your scripts is built into the core of Airflow using the powerful **Jinja** templating engine.
49 - **Scalable**: Airflow has a modular architecture and uses a message queue to orchestrate an arbitrary number of workers. Airflow is ready to scale to infinity.
