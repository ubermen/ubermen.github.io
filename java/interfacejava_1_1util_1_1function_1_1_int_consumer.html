Detailed Description

Represents an operation that accepts a single

int

-valued argument and returns no result. This is the primitive type specialization of Consumer for

int

. Unlike most other functional interfaces,

IntConsumer

is expected to operate via side-effects.

This is a functional interface whose functional method is accept(int).
