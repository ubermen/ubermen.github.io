Detailed Description

Indicates that an unknown kind of annotation value was encountered. This can occur if the language evolves and new kinds of annotation values can be stored in an annotation. May be thrown by an AnnotationValueVisitor annotation value visitor} to indicate that the visitor was created for a prior version of the language.
