int org.omg.CORBA.PolicyOperations.policy_type ( )

Returns the constant value that corresponds to the type of the policy object. The values of the policy objects are allocated by the OMG. New values for PolicyType should be obtained from the OMG by sending mail to reque.nosp@m.st@o.nosp@m.mg.or.nosp@m.g. In general the constant values that are allocated are defined in conjunction with the definition of the corresponding policy object.
