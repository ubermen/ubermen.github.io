Detailed Description

This is the superclass for Paints which use a multiple color gradient to fill in their raster. It provides storage for variables and enumerated values common to

LinearGradientPaint

and

RadialGradientPaint

.

Author
Nicholas Talian, Vincent Hardy, Jim Graham, Jerry Evans
