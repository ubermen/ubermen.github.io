Detailed Description

Use an adapter that implements XmlAdapter for custom marshaling.

Usage:

The annotation can be used with the following program elements:

  • a JavaBean property
  • field
  • parameter
  • package
  • from within XmlJavaTypeAdapters

When annotation is defined on a class, it applies to all references to the class.

When annotation is defined at the package level it applies to all references from within the package to .type().

When annotation is defined on the field, property or parameter, then the annotation applies to the field, property or the parameter only.

A annotation on a field, property or parameter overrides the annotation associated with the class being referenced by the field, property or parameter.

A annotation on a class overrides the annotation specified at the package level for that class.

This annotation can be used with the following other annotations: XmlElement, XmlAttribute, XmlElementRef, XmlElementRefs, XmlAnyElement. This can also be used at the package level with the following annotations: XmlAccessorType, XmlSchema, XmlSchemaType, XmlSchemaTypes.

Example: See example in XmlAdapter
