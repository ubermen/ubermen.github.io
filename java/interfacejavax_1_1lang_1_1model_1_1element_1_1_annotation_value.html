Detailed Description

Represents a value of an annotation type element. A value is of one of the following types:

  • a wrapper class (such as Integer) for a primitive type
  • String
  • TypeMirror
  • VariableElement
    (representing an enum constant)
  • AnnotationMirror
  • List<? extends AnnotationValue>
    (representing the elements, in declared order, if the value is an array)
