Detailed Description

This is a general support class to provide support for implementing the BeanContextChild protocol.

This class may either be directly subclassed, or encapsulated and delegated to in order to implement this interface for a given component.

Author
Laurence P. G. Cable

Report a vetoable property update to any registered listeners. If anyone vetos the change, then fire a new event reverting everyone to the old value and then rethrow the PropertyVetoException.

No event is fired if old and new are equal and non-null.

Parameters

oldValueThe old value of the property
