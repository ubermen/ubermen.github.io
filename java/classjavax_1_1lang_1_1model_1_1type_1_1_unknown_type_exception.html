Detailed Description

Indicates that an unknown kind of type was encountered. This can occur if the language evolves and new kinds of types are added to the

TypeMirror

hierarchy. May be thrown by a TypeVisitor type visitor} to indicate that the visitor was created for a prior version of the language.
