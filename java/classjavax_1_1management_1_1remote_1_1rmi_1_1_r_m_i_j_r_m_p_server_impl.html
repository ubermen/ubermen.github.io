Detailed Description

An RMIServer object that is exported through JRMP and that creates client connections as RMI objects exported through JRMP. User code does not usually reference this class directly.

See also
RMIServerImpl

Creates a new RMIServer object that will be exported on the given port using the given socket factories.

Parameters
portthe port on which this object and the RMIConnectionImpl objects it creates will be exported. Can be zero, to indicate any available port.
csfthe client socket factory for the created RMI objects. Can be null.
ssfthe server socket factory for the created RMI objects. Can be null.
envthe environment map. Can be null.
Exceptions
IOExceptionif the RMIServer object cannot be created.
IllegalArgumentExceptionif port is negative.

Creates a new client connection as an RMI object exported through JRMP. The port and socket factories for the new RMIConnection object are the ones supplied to the RMIJRMPServerImpl constructor.

Parameters
connectionIdthe ID of the new connection. Every connection opened by this connector server will have a different id. The behavior is unspecified if this parameter is null.
subjectthe authenticated subject. Can be null.
Returns
the newly-created RMIConnection.
Exceptions
IOExceptionif the new RMIConnection object cannot be created or exported.
