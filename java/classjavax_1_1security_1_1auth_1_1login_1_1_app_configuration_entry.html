Detailed Description

This class represents a single

LoginModule

entry configured for the application specified in the

getAppConfigurationEntry(String appName)

method in the

Configuration

class. Each respective

AppConfigurationEntry

contains a

LoginModule

name, a control flag (specifying whether this

LoginModule

is REQUIRED, REQUISITE, SUFFICIENT, or OPTIONAL), and LoginModule-specific options. Please refer to the

Configuration

class for more information on the different control flags and their semantics.

See also
javax.security.auth.login.Configuration
