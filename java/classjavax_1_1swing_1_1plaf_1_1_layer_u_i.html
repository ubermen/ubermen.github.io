Detailed Description

The base class for all javax.swing.JLayer's UI delegates.

paint(java.awt.Graphics, javax.swing.JComponent) method performs the painting of the

JLayer

and eventDispatched(AWTEvent, JLayer) method is notified about any

AWTEvent

s which have been generated by a

JLayer

or any of its subcomponents.

The

LayerUI

differs from the UI delegates of the other components, because it is LookAndFeel independent and is not updated by default when the system LookAndFeel is changed.

The subclasses of

LayerUI

can either be stateless and shareable by multiple

JLayer

s or not shareable.
