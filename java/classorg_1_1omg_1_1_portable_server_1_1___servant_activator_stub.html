This operation is invoked whenever a servant for an object is deactivated, assuming the POA has the USE_SERVANT_MANAGER and RETAIN policies.

Parameters
oidobject Id associated with the object being deactivated.
adapterobject reference for the POA in which the object was active.
servcontains reference to the servant associated with the object being deactivated.
cleanup_in_progressif TRUE indicates that destroy or deactivate is called with etherealize_objects param of TRUE. FALSE indicates that etherealize was called due to other reasons.
remaining_activationsindicates whether the Servant Manager can destroy a servant. If set to TRUE, the Servant Manager should wait until all invocations in progress have completed.
