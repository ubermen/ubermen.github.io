javax.xml.bind.annotation.XmlList Interface Reference

Detailed Description

Used to map a property to a list simple type.

Usage

The annotation can be used with the following program elements:

  • JavaBean property
  • field

When a collection property is annotated just with , each item in the collection will be wrapped by an element. For example,

&#64;XmlRootElement
class Foo {
    &#64;XmlElement
    List<String> data;
}

would produce XML like this:

<foo>
  <data>abc</data>
  <data>def</data>
</foo>

&#64;XmlList annotation, on the other hand, allows multiple values to be represented as whitespace-separated tokens in a single element. For example,

&#64;XmlRootElement
class Foo {
    &#64;XmlElement
    &#64;XmlList
    List<String> data;
}

the above code will produce XML like this:

<foo>
  <data>abc def</data>
</foo>

This annotation can be used with the following annotations: XmlElement, XmlAttribute, XmlValue, XmlIDREF.

  • The use of with XmlValue while allowed, is redundant since XmlList maps a collection type to a simple schema type that derives by list just as XmlValue would.

  • The use of with XmlAttribute while allowed, is redundant since XmlList maps a collection type to a simple schema type that derives by list just as XmlAttribute would.
