Detailed Description

The

OpenMBeanInfoSupport

class describes the management information of an open MBean: it is a subclass of javax.management.MBeanInfo, and it implements the OpenMBeanInfo interface. Note that an open MBean is recognized as such if its

getMBeanInfo()

method returns an instance of a class which implements the OpenMBeanInfo interface, typically

OpenMBeanInfoSupport

.
