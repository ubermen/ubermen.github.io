Detailed Description

A date in the Minguo calendar system.

This date operates using the MinguoChronology Minguo calendar}. This calendar system is primarily used in the Republic of China, often known as Taiwan. Dates are aligned such that

0001-01-01 (Minguo)

is

1912-01-01 (ISO)

.

This is a value-based class; use of identity-sensitive operations (including reference equality (

==

), identity hash code, or synchronization) on instances of

MinguoDate

may have unpredictable results and should be avoided. The

equals

method should be used for comparisons.

This class is immutable and thread-safe.
