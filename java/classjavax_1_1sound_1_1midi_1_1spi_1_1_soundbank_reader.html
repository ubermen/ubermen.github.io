Detailed Description

A

SoundbankReader

supplies soundbank file-reading services. Concrete subclasses of

SoundbankReader

parse a given soundbank file, producing a javax.sound.midi.Soundbank object that can be loaded into a javax.sound.midi.Synthesizer.
